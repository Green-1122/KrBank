# KrBank - Premium Online Banking Platform

A **complete, production-grade** online banking system built with pure PHP 8+ and modern JavaScript.

![KrBank](https://via.placeholder.com/800x400/1e40af/ffffff?text=KrBank)

## ✨ Features

### **User Features**
- Beautiful responsive dashboard with real-time balance
- Multiple account types (Checking, Savings, Investment)
- Local, International & Crypto Transfers
- Virtual & Physical Cards (Visa, Mastercard, Amex)
- Loans Management & Application
- Stock Trading & Portfolio
- Savings Goals with progress tracking
- Cryptocurrency Send / Receive / Swap
- PDF Account Statements
- Multi-channel Notifications (Email + Web Push + FCM)

### **Admin Panel**
- Full user management with backdating
- Transaction oversight and editing
- Card & Loan management
- Feature toggles (COT, IMF, Tax Code)
- VAPID key rotation
- System statistics

### **Security & Best Practices**
- PDO Prepared Statements (No SQL Injection)
- CSRF Protection
- Session Regeneration
- Input Sanitization
- VAPID Key Rotation for Push Notifications
- Dark/Light Mode with persistence

---

## 🛠 Tech Stack

- **Backend**: PHP 8+ (OOP, PDO)
- **Frontend**: HTML5, Tailwind CSS, Vanilla JS + Chart.js
- **Database**: MySQL
- **Notifications**: PHPMailer + Web Push + Firebase Cloud Messaging (FCM)
- **PDF**: TCPDF
- **Push**: Service Worker + VAPID
